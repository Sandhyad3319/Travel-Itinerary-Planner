# Travel Itinerary Planner

## Project Overview

The Travel Itinerary Planner is a web-based application that helps users organize and manage their travel plans efficiently. It allows users to create travel itineraries, add destinations, and plan trip schedules in a simple and structured way.

## Features

* Create and manage travel itineraries
* Add and organize destinations
* Plan trip schedules
* User-friendly interface for easy navigation

## Technologies Used

* Python
* Flask
* HTML
* CSS

## Installation

1. Clone the repository:

```
git clone https://github.com/Sandhyad3319/travel-itinerary-planner.git
```

2. Navigate to the project folder:

```
cd travel-itinerary-planner
```

3. Install the required dependencies:

```
pip install -r requirements.txt
```

4. Run the application:

```
python app.py
```

5. Open your browser and go to:

```
http://127.0.0.1:5000
```
